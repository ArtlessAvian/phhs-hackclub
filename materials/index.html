So you want to learn code?

Here are some resources for you to use!

Our Tutorials

Our team members have personally made comprehensive, in-depth tutorials fit for coders of all levels!
Tutorials include short guides on how to make cool applications, lessons on the basics
of programming languages, and crash courses of computer-science-related topics.

Hack Club Workshops

The official Hack Club website has provided a number of free, interactive workshops made by
the main Hack Club team and members of the the Hack Club community around the world.
Workshops include tutorials on how to make your own browser games, chat apps, and more!

Coding Exercises

We believe the best way to learn code is through hands-on practice, so our team suggests that
anyone interested in learning a basic programming language should visit CodingBat, a website
consisting of coding exercises in Java and Python. The exercises range from beginner-level to intermediate.
